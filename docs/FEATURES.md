# ProVision WorkSuite - Feature Guide

This document provides a detailed overview of all features and modules available in ProVision WorkSuite.

---

## 1. Dashboard & Analytics

The dashboard provides a high-level overview of project health and team productivity.

- **Project Statistics**: Real-time count of active, completed, and delayed projects.
- **Team Workload**: Visual breakdown of task distribution across the team.
- **Recent Activity**: A live feed of all significant changes within the platform.
- **Productivity Metrics**: Charts showing task completion rates and milestone progress.

## 2. Project Management

Efficiently organize and track complex initiatives.

- **Project Creation**: Define scope, budget, deadlines, and ownership.
- **Milestones**: Break projects into key phases with specific target dates.
- **Budget Tracking**: Monitor financial health and resource allocation.
- **Document Management**: Upload and link essential files directly to projects.

## 3. Task Management (Kanban)

A visual drag-and-drop system for daily operations.

- **Status Columns**: Standardized "To Do", "In Progress", and "Done" workflow.
- **Task Details**: Rich text descriptions, priority levels, and tag management.
- **Assignments**: Single or multiple team members per task.
- **Time Tracking**: Log hours spent on specific tasks for accurate billing and reporting.
- **Recurring Tasks**: Automate repetitive work with powerful scheduling options.

## 4. Real-Time Communication

Seamless collaboration without leaving the platform.

- **Group Chat**: Persistent rooms for projects and departments.
- **Direct Messaging**: Private one-on-one conversations.
- **File Sharing**: Integrated file uploads within the chat interface.
- **Members Directory**: Quick access to team profiles, roles, and status.

## 5. Video Conferencing (ZegoCloud)

Professional-grade video meetings integrated directly into the workflow.

- **One-Click Calls**: Start meetings from any chat or project room.
- **Managed Infrastructure**: Powered by ZegoCloud for low-latency global connections.
- **In-Call Features**: Screen sharing, user management, and layout controls.
- **Client Credentials**: Admins can use their own ZegoCloud quota via the Settings panel.

## 6. Planning & Calendar

Stay ahead of deadlines with a centralized view.

- **Unified Timeline**: See projects, milestones, and tasks in a single view.
- **Event Scheduling**: Create meetings and sync them with project deadlines.
- **Drag-to-Reschedule**: Quickly adjust dates directly on the calendar interface.

## 7. Advanced Administration

Enterprise-grade control over the entire platform.

- **User Management**: Invite members, define roles, and manage permissions.
- **Dynamic Configuration**: Change database and video credentials without redeploying.
- **Security Logs**: Monitor login activity and sensitive system changes.
- **Export Engine**: Generate professional PDF, Excel, and CSV reports.
- **Theme Customization**: Align the workspace with your brand through color presets and dark mode.

---

> [!TIP]
> Use the **Global Search** (Cmd/Ctrl + K) to quickly jump between projects, tasks, and team members from anywhere in the app.
